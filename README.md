# Recommendation System

Teknik-teknik sistem rekomendasi menggunakan jcop_recsys. Sumber materi: [WiraDKP](https://github.com/WiraDKP/recommendation_system)

## Instalasi Environment

- Jalankan command ini untuk menginstall environment
  ```
  conda env create -f env_jcop_recsys.yml
  ```
- Jalankan command berikut untuk mengecek instalasi
  ```
  python check_installation.py
  ```
